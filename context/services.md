## SHOPIFY DEVELOPMENT

### Shopify Theme Build
A production-ready Shopify store built on a proven theme, customised to the client's brand. Three tiers based on complexity.

**What's included (all tiers):**
- Phase 1 (MVP Build): Product setup & configuration, theme setup with brand colours/typography/logo/imagery, all core pages created and populated (Home, About, Contact, FAQ, Collection, Product), customer journey setup (navigation, CTAs, cart behaviour, upsells), mobile-first responsive build across all devices
- Phase 2 (Optimisation & Config): Client feedback addressed, basic CRO (trust signals, CTA placement, checkout flow), basic SEO (metadata, alt tags, heading structure), payment gateway config (PayFast, Peach Payments, Yoco), delivery & logistics setup (Bob Go, Pargo, or manual rates), newsletter popup + Omnisend integration
- Phase 3 (QA & Launch): Legal policy pages (T&Cs, Privacy, Delivery & Returns), branded email notification templates, page speed optimisation (>80 PageSpeed Insights), pre-launch QA (cross-browser, device testing, checkout simulation), go-live + post-launch smoke test + handover

**Timeline:** 4–6 weeks
**Payment terms:** 50% upfront / 30% after Phase 2 / 20% on launch

**When to recommend:** Client needs a new Shopify store and has brand assets ready but no Figma design files. Ideal for new-to-Shopify brands or those rebuilding an existing store on a solid theme foundation.

### WooCommerce → Shopify Migration
Full migration from WooCommerce to Shopify with zero data loss, minimal downtime, and preserved SEO equity.

**What's included:**
- Pre-migration audit & planning: WooCommerce site audit (products, collections, pages, customers, orders, apps, integrations), URL & SEO mapping with 301 redirect plan, content & data inventory, phased migration plan with rollback strategy
- Phase 1 (Data Migration & Store Build, 2–4 weeks): All products migrated with variants, images, pricing, inventory, tags; customer data migration (accounts, addresses, order history); page & content migration; theme setup matching existing design
- Phase 2 (Config, Redirects & Testing, 1–2 weeks): 301 redirect implementation, payment gateway setup, shipping & logistics config, app migration, basic SEO verification, staging review
- Phase 3 (QA & Cutover, 1 week): Final data sync, cross-browser QA, checkout testing, DNS cutover & launch, 48-hour post-launch monitoring, WooCommerce decommission guidance

**Timeline:** 4–8 weeks
**Payment terms:** 50% upfront / 30% after Phase 2 / 20% on launch

**When to recommend:** Client is currently on WooCommerce and wants to move to Shopify. Key selling points are zero data loss, SEO equity preservation via 301 redirects, and a clean handover.
